# Netflix Data Analysis

## Objective
Analyze Netflix Movies and TV Shows dataset and generate insights using Python.

---

## Dataset
Dataset Name:
netflix_titles.csv

Source:
Netflix Movies and TV Shows Dataset

---

## Tools Used

- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Jupyter Lab

---

## Analysis

1. Data Cleaning
2. Missing Values Handling
3. Duplicate Removal
4. Movies vs TV Shows Analysis
5. Release Year Analysis
6. Country Analysis
7. Genre Analysis
8. Duration Analysis

---

## Insights

- Movies are more than TV Shows
- USA produced the highest content
- Content growth increased during recent years
- Drama was among the top genres
